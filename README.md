# The-Smart-Grade-Calculator(Java)
A robust Java-based tool designed to help students manage their academic performance through weighted calculations and predictive "what-if" scenarios.

## Features
-**Weighted Average Calculation:**
Handles different categories like Finals, Midterms, and Homework.
-**What-If Analysis:**
Calculates the minimum score needed on upcoming assignments to reach a target grade.
-**GPA Projection:**
Estimates based on current course performance.
-**Data Persistence:**
Save your course data to a local Text file
